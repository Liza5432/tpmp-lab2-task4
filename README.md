
## Overview
Данный проект представляет собой консольное приложение на языке C для 
управления базой данных покупателей. Программа разработана в рамках 
лабораторной работы №2 и демонстрирует работу с файловым вводом-выводом, 
строковыми функциями и структурами данных.

## Usage
Проект организован согласно модели КИС, разделяющей интерфейсы, реализацию 
и исполняемый код.

### Файловая структура:
* `include/buyer.h` — Заголовочный файл с прототипами функций.
* `src/buyer.c` — Реализация функций работы с данными.
* `src/main.c` — Точка входа в программу.
* `Makefile` — Сценарий для автоматизированной сборки.

### Сборка и запуск:
1. **Сборка**: Выполните команду `make` в корневой директории проекта. 
Процесс включает компиляцию исходных файлов `.c` с использованием `gcc` и 
флагов `-Wall -Wextra`.
2. **Запуск**: После успешной сборки запустите исполняемый файл командой 
`./app`.
3. **Очистка**: Для удаления объектных файлов и приложения используйте 
`make clean`.

## Additional Notes
* **Данные**: Программа генерирует файл `data.txt`, содержащий записи о 
покупателях (ФИО, пол, национальность, адрес, данные карты и т.д.).
* **Фильтрация**: Результаты поиска по городу Брест сохраняются в файл 
`result.txt`.
